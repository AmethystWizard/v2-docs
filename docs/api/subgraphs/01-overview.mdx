# Sablier Subgraphs

Sablier uses multiple [subgraphs](https://thegraph.com/docs/about/introduction#what-the-graph-is) for indexing and
organizing data from the Sablier smart contracts. These subgraphs are hosted on The Graph Network, as well as on the The
Graph Hosted Service, and can be used to query Sablier data.
